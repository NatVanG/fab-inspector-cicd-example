To create a PBI Inspector V2 (aka Fab Inspector) rule from user instructions please follow the guidelines and documentation at https://github.com/NatVanG/PBI-InspectorV2/wiki and https://github.com/NatVanG/PBI-InspectorV2/blob/main/README.md, and also consider example rules at https://raw.githubusercontent.com/NatVanG/PBI-InspectorV2/refs/heads/main/Rules/Base-rules.json and https://raw.githubusercontent.com/NatVanG/PBI-InspectorV2/refs/heads/main/DocsExamples/Examples-rules.json. Please remember that a rule in this context should be valid JSON within a "rules" JSON array. Most likely the rules array is already present in the file so just add a new item to the array.
